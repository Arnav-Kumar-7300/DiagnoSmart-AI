# 🧠 DiagnoSmart AI

### Intelligent Diagnostic Report Analyzer

---

## 🚀 Overview

**DiagnoSmart AI** is an intelligent healthcare assistant that analyzes diagnostic test values and provides meaningful insights such as **disease risk, severity level, and health recommendations**.

This project is designed to help users understand their medical reports in a simple and interactive way before consulting a doctor.

---

## 🎯 Key Features

- ✅ Supports Multiple Diagnostic Tests (10 Tests)
- ✅ Displays Normal Ranges with Units
- ✅ Predicts Possible Disease Risks
- ✅ Determines Severity Level (Normal / Mild / Moderate / High)
- ✅ Provides Tips, Precautions & Remedies
- ✅ Smart Doctor Consultation Suggestions
- ✅ User-Friendly Interface (Streamlit आधारित)

---

## 🧪 Supported Tests

* 🩸 Blood Test (CBC)
* 🧬 Liver Function Test (LFT)
* 💧 Kidney Function Test (KFT)
* 🍬 Diabetes Test
* ❤️ Lipid Profile
* 🦋 Thyroid Test
* 🚽 Urine Test
* 🦴 Bone Health Test
* 💓 Blood Pressure
* 🖥️ Sonography (Basic Indicators)

---

## ⚙️ How It Works

1. Select a diagnostic test
2. View normal parameter ranges
3. Enter your report values
4. Click **Analyze**
5. Get:

   * 🧠 Predicted Disease Risk
   * ⚠️ Severity Level
   * 💡 Tips & Precautions
   * 👨‍⚕️ Doctor Recommendation

## ▶️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Arnav-Kumar-7300/DiagnoSmart-AI
cd DiagnoSmart-AI
```

---

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Run the Model Training

```bash
python train.py
```

---

### 4️⃣ Launch the Application

```bash
streamlit run app.py
```
